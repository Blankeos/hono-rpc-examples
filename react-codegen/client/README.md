# react-codegen (client)

### Get Started

1. Make sure `react-codegen/server` is running.
2. Setup .env (this contains your Backend URL)
   ```sh
   cp .env.example .env
   ```
3. Generate API types (Do it everytime backend has changes)
   ```sh
   bun run generate:api
   ```
4. Install deps and run server.

   ```sh
   bun dev
   ```
